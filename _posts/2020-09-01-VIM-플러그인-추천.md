필자가 자주 사용하는 빔 플러그인을 추천하고자 한다.

서
-

```c
Plug 'scrooloose/nerdtree'

Plug 'vim-airline/vim-airline'
Plug 'bagrat/vim-buffet'
Plug 'junegunn/fzf'
Plug 'rking/ag.vim'
Plug 'majutsushi/tagbar'
Plug 'will133/vim-dirdiff'
Plug 'tpope/vim-unimpaired'
Plug 'sareyko/neat.vim'
Plug 'mhinz/vim-startify'
Plug 'easymotion/vim-easymotion'
Plug 'MattesGroeger/vim-bookmarks'
Plug 'tpope/vim-surround'
Plug 'Raimondi/delimitMate'
if has ('nvim')
    Plug 'ryanoasis/vim-devicons'
endif
Plug 'tiagofumo/vim-nerdtree-syntax-highlight'
Plug 'scrooloose/nerdcommenter'
Plug 'terryma/vim-multiple-cursors'
Plug 'frazrepo/vim-rainbow'
Plug 'christoomey/vim-tmux-navigator'
Plug 'vim-scripts/vcscommand.vim'
```

에디터 UI
------

### nerdtree

### vim-nerdtree-syntax-highlight

너드트리 리스트에 하이라이트 효과를 준다.

### vim-devicons

너트트리에 파일 아이콘을 추가한다. (폰트와 터미널앱이 합자(ligature)를 지원해야한다.

### vim-airline

### vim-buffet

![1.png](/assets/img/resources/23414348600B662791A9FEBE8D9E111C.png)

버퍼를 상단바로 보여준다.

### tagbar

### vim-rainbow

괄호의 짝끼리 폰트색상을 변경해서, 코드가독성을 높인다.

### vim-tmux-navigator

빔 사용 중에 tmux 패널을 단축키로 이동할 수 있다.

에디터 기능
------

### vim-startify

![2.png](/assets/img/resources/279E49A458EAC48E697573602B6EF71E.png)

최근에 연 파일 리스트를 보여주고, 선택해서 열 수 있다.

세션을 지정해서 작업하고 있는 워크스페이스로 바로 이동할 수 있다.

### vim-bookmarks

간단한 북마크 기능, 북마크를 활성화 할 경우 파란색 깃발이 표시된다.

코드편집
----

### vim-easymotion

![3.png](/assets/img/resources/91ACFB7F56B89BA71EDE0A649149EB68.png)

화면에서 보이는 문자중에서 원하는 문자로 커서를 이동할 수 있다. 매우 편리

### vim-surround

### delimitMate

짝괄호를 자동으로 생성해준다.

### nerdcommenter

주석을 삽입해준다.

### vim-multiple-cursors

![4.png](/assets/img/resources/F159FD2343C225FE5D66A6A94D00F936.png)

동일한 단어를 선택해서 한번에 편집할 수 있게 해준다.

위에 스크린샷은 set 스트링을 5번재 줄에서 아래로 4개 선택하여 text로 한번에 교체하는 중이다.

검색
--

### fzf

현재 위치에서 재귀적으로 파일을 퍼지검색한다.

### ag.vim

![5.png](/assets/img/resources/FE70FCB5961050C7F457DE9980AD5F3D.png)

현재 위치에서 재귀적으로 파일의 문자열을 검색한다. 검색 결과는 quick-fix로 보여준다.

비교도구
----

### vim-dirdiff

beyond-compare처럼 디렉토리를 비교한다.

### vcscommand.vim

cvs, svn, git 등 형상관리도구를 통합한 명령어를 제공한다. 

정렬도구
